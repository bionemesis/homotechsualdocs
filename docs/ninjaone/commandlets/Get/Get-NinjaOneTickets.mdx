## SYNOPSIS

Gets tickets from the NinjaOne API.

## SYNTAX

### Single

```powershell
Get-NinjaOneTickets -ticketId <Int32> [<CommonParameters>]
```

### Board

```powershell
Get-NinjaOneTickets -boardId <String> [-filters <Hashtable>] [-includeMetadata] [<CommonParameters>]
```

## DESCRIPTION

Retrieves tickets from the NinjaOne v2 API.
