## SYNOPSIS

Gets agent installer URL from the NinjaOne API.

## SYNTAX

```powershell
Get-NinjaOneInstaller [-organisationID] <Int32> [-locationID] <Int32> [-installerType] <String>
 [<CommonParameters>]
```

## DESCRIPTION

Retrieves agent installer URL from the NinjaOne v2 API.
