## SYNOPSIS

Sets a new maintenance window for the specified device(s)

## SYNTAX

```powershell
Set-NinjaOneDeviceMaintenance [-deviceId] <String> [-disabledFeatures] <String[]> [[-start] <DateTime>]
 [[-unixStart] <Int32>] [[-end] <DateTime>] [[-unixEnd] <Int32>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Create a new maintenance window for the given device(s) using the NinjaOne v2 API.
