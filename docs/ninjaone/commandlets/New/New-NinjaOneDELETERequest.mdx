## SYNOPSIS

Builds a request for the NinjaOne API.

## SYNTAX

```powershell
New-NinjaOneDELETERequest [-Resource] <String> [<CommonParameters>]
```

## DESCRIPTION

Wrapper function to build web requests for the NinjaOne API.

## EXAMPLES

### EXAMPLE 1

```powershell
Cancel the maintenance for device with id 1.
```

PS ~&gt; New-NinjaOneDELETERequest -Resource "/v2/device/1/maintenance"
