## SYNOPSIS

Builds a request for the NinjaOne API.

## SYNTAX

```powershell
New-NinjaOnePATCHRequest [-Resource] <String> [[-QSCollection] <Hashtable>] [-Body] <Hashtable>
 [<CommonParameters>]
```

## DESCRIPTION

Wrapper function to build web requests for the NinjaOne API.

## EXAMPLES

### EXAMPLE 1

```powershell
Make a PATCH request to the custom fields endpoint for device 1.
```

PS C:\\&gt; New-NinjaOnePATCHRequest -Resource "/v2/device/1/custom-fields" -Body @{"myCustomField" = "value"}
