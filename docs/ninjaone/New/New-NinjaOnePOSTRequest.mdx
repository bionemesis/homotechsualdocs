## SYNOPSIS

Builds a request for the NinjaOne API.

## SYNTAX

```powershell
New-NinjaOnePOSTRequest [-Resource] <String> [[-QSCollection] <Hashtable>] [[-Body] <Hashtable>]
 [<CommonParameters>]
```

## DESCRIPTION

Wrapper function to build web requests for the NinjaOne API.

## EXAMPLES

### EXAMPLE 1

```powershell
New-NinjaOnePOSTRequest -Method "POST" -Resource "/v2/organizations"
```
