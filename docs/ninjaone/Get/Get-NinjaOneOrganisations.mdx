## SYNOPSIS

Gets organisations from the NinjaOne API.

## SYNTAX

### Multi (Default)

```powershell
Get-NinjaOneOrganisations [-pageSize <Int32>] [-after <Int32>] [-detailed] [<CommonParameters>]
```

### Single

```powershell
Get-NinjaOneOrganisations -organisationId <Int32> [<CommonParameters>]
```

## DESCRIPTION

Retrieves organisations from the NinjaOne v2 API.
