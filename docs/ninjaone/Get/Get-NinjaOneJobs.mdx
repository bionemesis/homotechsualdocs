## SYNOPSIS

Gets jobs from the NinjaOne API.

## SYNTAX

```powershell
Get-NinjaOneJobs [[-jobType] <String>] [[-deviceFilter] <String>] [[-languageTag] <String>]
 [[-timeZone] <String>] [-deviceID] <Int32> [<CommonParameters>]
```

## DESCRIPTION

Retrieves jobs from the NinjaOne v2 API.
