## SYNOPSIS

Gets activities from the NinjaOne API.

## SYNTAX

```powershell
Get-NinjaOneActivities [[-deviceID] <Int32>] [[-class] <String>] [[-before] <DateTime>] [[-after] <DateTime>]
 [[-olderThan] <Int32>] [[-newerThan] <Int32>] [[-type] <String>] [[-status] <String>] [[-user] <String>]
 [[-seriesUid] <String>] [[-deviceFilter] <String>] [[-pageSize] <Int32>] [[-languageTag] <String>]
 [[-timeZone] <String>] [<CommonParameters>]
```

## DESCRIPTION

Retrieves activities from the NinjaOne v2 API.
