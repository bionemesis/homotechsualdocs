## SYNOPSIS

Gets locations from the NinjaOne API.

## SYNTAX

```powershell
Get-NinjaOneLocations [[-pageSize] <Int32>] [[-after] <Int32>] [-organisationID] <Int32> [<CommonParameters>]
```

## DESCRIPTION

Retrieves locations from the NinjaOne v2 API.
